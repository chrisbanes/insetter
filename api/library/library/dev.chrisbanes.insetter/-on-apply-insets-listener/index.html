Listener for applying window insets on a view in a custom way.

This class is similar to to androidx.core.view.OnApplyWindowInsetsListener but contains a third parameter on onApplyInsets. It is designed to be used in conjunction with Insetter.Builder.setOnApplyInsetsListener.
