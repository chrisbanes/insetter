Bit mask of WindowInsetsCompat.Types to apply as padding. The windowInsetTypesOf function is useful for creating the bit mask.

sides

Bit mask of Sides containing which sides to apply. Defaults to Side.ALL to apply all sides. The mask can be created via sidesOf.

Whether we should animate the padding whilst a window insets animation with the type is ongoing.
